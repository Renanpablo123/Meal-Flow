O HTML que significa HyperText Markup Language é basicamente a estrutura de qualquer site que a gente vê por aí. Uma coisa importante que o professor falou é que o HTML não é uma linguagem de programação porque ele não tem lógica ou cálculos, ele é só uma linguagem de marcação pra falar pro navegador o que é cada coisa na tela, tipo avisar que tal texto é um título e tal imagem deve aparecer ali. É como se fosse o esqueleto da página antes de colocar as cores com o CSS.

Sobre as tags que a gente viu, as principais são o h1 que é o título maior e vai até o h6 que é o menor de todos pra fazer a hierarquia do texto. O p serve pra escrever parágrafos normais e a tag a é pra fazer os links usando o href pra dizer pra onde o link vai. Para as imagens a gente usa a tag img com o src que é o caminho do arquivo e o alt que é aquele texto que aparece se a imagem não carregar. Também tem o ul e o li pra fazer aquelas listas com bolinhas que a gente sempre usa.

A tag div é uma das mais importantes porque ela funciona como uma caixa invisível. Ela serve pra separar os pedaços do site e organizar tudo. A gente usa a div pra fazer o aninhamento que é basicamente colocar uma tag dentro da outra, tipo colocar um título e um parágrafo dentro de uma div pra eles ficarem juntos. Isso ajuda muito na hora de organizar o código e depois pra aplicar estilos em um grupo inteiro de elementos de uma vez só, deixando tudo bem mais limpo e fácil de entender.
