# Meal-Flow
O MealFlow é uma aplicação web para gestão de pedidos em restaurantes de pequeno e médio porte, usada apenas por funcionários. Digitaliza e organiza o atendimento com pedidos de ID único, reduz erros de anotações manuais e oferece uma solução simples, acessível, modular e escalável.
Projeto desenvolvido para a atividade do LabSIS.
